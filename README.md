# Typing Over It 랜딩

**Typing Over It** 랜딩은 실제 제품인 `ICE0208/typing-over-it`의 내용을 기준으로 정리한 소개 페이지입니다. 제품은 VS Code처럼 보이는 웹 IDE에서 사용자의 타이핑 패턴을 감지하고, 에디터 하단에 *Getting Over It* 느낌의 KO/EN 자막 오버레이를 띄웁니다.

## 현재 제품 기준 핵심

- VS Code풍 웹 IDE 화면
- Monaco 에디터, 파일 트리, 탭, 저장 상태, 변경 보존 흐름
- Panic Typing, Backspace Spiral, Frozen Silence, Key Repeat 감지
- 에디터 하단 중앙의 한국어 + EN 2줄 자막 오버레이
- 자막은 fade-in, hold, fade-out 리듬으로 표시
- 랜딩의 EN 라인은 정상 번역이 아니라 일부러 이상한 해석본 톤으로 작성

## 랜딩 톤

제품 사실은 정확히 전달하되, 문구는 너무 설명서처럼 굳지 않게 유지합니다. 내부 배포 방식, 개발용 단축키, 운영용 주석처럼 사용자에게 보일 필요 없는 내용은 랜딩 카피에 넣지 않습니다.
